# Contributing

Thanks for your interest in Thorm.

## Current Status

Thorm is not currently open to external code contributions.

At this stage, the priority is stabilizing the public API, packaging, documentation, and release workflow before opening the codebase to outside implementation work.

## What Is Welcome

The following feedback may still be useful:

- bug reports
- documentation corrections
- API ergonomics feedback
- example issues or unclear behavior reports

## Security Issues

If you believe you found a security issue, do not open a public issue.

Follow the process in [SECURITY.md](SECURITY.md).

## Future Changes

This policy may change once Thorm reaches a more stable public development phase.
